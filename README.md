# 🚀 Antigravity – Englabs Projects

## 📌 Overview
**Antigravity** is the professional project management workflow for **Englabs India Pvt. Ltd.**, engineered to ensure absolute transparency, efficiency, and real-time tracking of engineering and fabrication projects.

## 🏗️ Project Identification (CXXX)
Every project within the system is uniquely identified using a standardized coding format:
- **Format**: `CXXX` (e.g., `C001`, `C002`, `C101`)
- **Storage**: All project-related documentation is systematically organized within a centralized **OneDrive** repository.

## 🔄 The Workflow Lifecycle

### 1. Initiation & Planning
- **Quotation Generation**: Initial client proposal.
- **PO Confirmation**: Formalization of the project.
- **Detailed Planning**: Capturing Project Value, Delivery Terms (**ToPay** or **Porter** basis), and Allocated Budget.

### 2. Production Tracking (Stages)
Projects move through defined production milestones:
1. **Engineering Design**: CAD/CAM and technical specifications.
2. **Machine Processing**: Primary CNC/Machining operations.
3. **Workshop Fabrication**: Structural assembly and welding.
4. **Sanding**: Surface preparation.
5. **Painting**: Primary coating.
6. **Drying**: Curing and stabilization.
7. **Finishing**: Final touch-ups and quality checks.
8. **Final Packaging**: Preparation for delivery.

### 3. Granular Operational Metrics
The system records part-wise details for every component:
- **Component Count**: Number of unique parts per project.
- **Manufacturing Time**: Precise duration for each stage.
- **Material Consumption**: Raw material usage tracking.
- **Workforce Involvement**: Employee assignment and accountability at each stage.

## 📊 Real-Time Dashboard
A centralized mission control provides immediate visibility into:
- **Project Status**: Current stage and progress percentage.
- **Financials**: Budget utilization vs. project value.
- **Operational Progress**: Bottleneck identification and efficiency metrics.
- **Delivery Readiness**: Real-time logistics status.
